# Sewanee Transit

Live shuttle tracking for the University of the South. Built originally
for CSCI 284 (Spring 2026); currently being modernized to a FastAPI +
PostGIS + React stack.

## Status

🚧 Modernization in progress. See branches for active work.

- `main` : stable
- `phase-1-backend` : FastAPI + PostgreSQL/PostGIS rewrite
- `phase-2-frontend` : Vite + React + TypeScript frontend (later)

## Original stack

Flask · MySQL · Jinja2 · Leaflet · plain JS

## Target stack

FastAPI · async SQLAlchemy 2 · PostgreSQL/PostGIS · WebSockets ·
Argon2id + JWT · Vite · React · TypeScript · PWA
